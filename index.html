FAQ

What is PUMP DOGE?

PUMP DOGE is a community-focused Token on the Binance Smart Chain. It's for everyone who loves Hinata and Karasuno and wants to support him on his way to the championships!

What kind of token is it?

It is an ENMT-Token, created with Unicrypt's token minter. This means, that there are no mint functions, no owner or admin functions, no unsafe code in the contract itself and its fully ERC20 compliant and fully decentralised.

What is the goal?

PUMP DOGE aims to be the number one Haiykuu token on the Binance Smart Chain. If PUMP DOGE is successful, there will be follow up tokens for all of the other amazing characters in the universe! And by being part of the community, you can decide which character will be next!

But is it SAFU?

Yeah it 100% is SAFU. Since it is an ENMT-Token, there is no way the project owners can run away with the liquidity. Additionally, there is a solidproof KYC, which will be revealed in the impossible event of a rug! This is as safe as it gets!
